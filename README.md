# Statistics Project 2

This project applies statistical inference techniques to real-world datasets using Python.

## Topics Covered

- Method of Moments (MOM) Estimation
- Maximum Likelihood Estimation (MLE)
- Confidence Intervals for Variance
- Confidence Intervals for Difference of Means
- Hypothesis Testing for Bernoulli Data

## Datasets Used

- Steps Tracker Dataset
- Weight-Height Dataset
- Loan Prediction Dataset

## Key Results

### Part 1: Gamma Distribution Estimation
Estimated Gamma distribution parameters for water intake data using MOM and MLE.

### Part 2: Confidence Interval for Variance
Computed a 95% confidence interval for the variance of water intake.

### Part 3: Difference of Means
Constructed a 95% confidence interval for the difference between male and female weights.

### Part 4: Hypothesis Testing
Performed a one-sided hypothesis test on loan approval data and showed that the approval rate is significantly greater than 50%.
